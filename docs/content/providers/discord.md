# <img src="../images/icon.png" width="16"> Discord

We can access the [Discord Developer Console](https://discord.com/developers/applications) 

Follow these steps to configure your app:

- Create a new application.
- Open OAuth2 --> General.
- Copy the Client ID.
- (Optional) Copy the Client Secret.
- Add your redirect URI.

## Notes

- Supports custom URI schemes.
- PKCE is enabled.

Default scopes: identify, email.
