# <img src="../images/icon.png" width="20"> Classes

This section documents the GDScript interface classes implemented and exposed by the plugin.

| Name | Description |
|---|---|
| OAuth2 | Main node that manages OAuth authorization flows, token exchange, and session storage. |
| OAuth2Config | Provides provider presets and configuration helpers for OAuth2 providers. |
| ProviderConfig | Encapsulates configuration for OAuth2 providers. |
